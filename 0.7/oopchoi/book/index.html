  • 뷰포트가 적용된 이미지

  • 미디어쿼리 적용하여 가로사이즈에 따라 배경색을 변경한다.

  • 부모 컨테이너 박스의 사이즈를 기준으로 자식 컨텐츠의 사이즈에 대한 비율 구하고 리사이즈 시 비율을 유지 한다.

  • 부모 컨테이너 박스의 사이즈를 기준으로 첫번째 자식 박스의 우측 마진에 대한 비율 구하고 리사이즈 시 비율을 유지 한다.

  • 부모 컨테이너 박스의 사이즈를 기준으로 좌우 패딩에 대한 비율 구하고 리사이즈 시 비율을 유지 한다.

  • 부모 컨테이너 박스의 사이즈를 기준으로 좌우 패딩에 대한 비율 구하고 리사이즈 시 비율을 유지 한다. 단, 600px를 넘지 않아야 한다.

  • calc CSS함수를 사용해 왼쪽 오른쪽 마진을 고정 시킨다.

  • 가변 패딩 직접 적용 (전체가로 : 960, div 가로:왼쪽300,오른쪽300, 패딩:왼쪽100, 오른쪽260)

  • 부모 폰트를 6em으로 두고 자식 폰트를 4em으로 지정하고 싶을 때 4em에 대해 비율을 구해야 한다. (1em은 16px, 즉 64/96=0.666666..)

  • 부모 폰트를 5em으로 두고 자식 폰트를 5rem으로 지정한다. rem은 html태그를 기준으로 계산하기 때문에 부모의 5em을 상속받지 않는다.

  • vw 적용 (브라우저의 너비를 100으로 기준하여 계산 된 px: vw값 * 브라우저너비px / 100 )

  • vh 적용 (브라우저의 높이를 100으로 기준하여 계산 된 px: vh값 * 브라우저높이px / 100 )

  • vmin 적용 (브라우저의 너비나 높이중 작은 값을 100으로 기준하여 계산 된 px: vmin값 * 작은px / 100 )

  • vmax 적용 (브라우저의 너비나 높이중 큰 값을 100으로 기준하여 계산 된 px: vmax값 * 큰px / 100 )

  • 동영상과 이미지는 width와 max-width를 100%로 사이즈를 지정하면 된다.

  • 미디어쿼리 min-width를 사용하여 박스 크기 비율 적용 (min-width는 작은값에서 큰값으로, max-width는 큰값에서 작은값으로)

  • 미디어쿼리 min-width를 사용하여 여러 개의 박스 크기 비율 적용

  • 플렉서블 박스 display:flex 적용 - 플렉서블 박스 설정

  • 플렉서블 박스 flex-direction 적용 - 배치 방향 설정 (row, row-reverse, column, column-reverse)

  • 플렉서블 박스 flex-wrap 적용 - 여러줄 배치 설정 (nowrap 기본, wrap, wrap-reverse 아래에서 위로 우에서 좌로)

  • 플렉서블 박스 flex-flow 적용 - 배치 방향과 여러줄 배치 설정 한번에 하기 ([flex-direction] [flex-wrap])

  • 플렉서블 박스 justify-content 적용 - 주축 방향으로 통합 배치 (flex-start 기본값, flex-end, center, space-between, space-around)

  • 플렉서블 박스 align-items 적용 - 교차축 방향으로 통합 배치 (stretch 기본값, flex-start, flex-end, center)

  • 플렉서블 박스 align-self 적용 - 교차축 방향으로 개별적 배치 (auto 부모 박스에 지정된 align-items 상속, stretch, flex-start, flex-end, center)

  • 플렉서블 박스 align-content 적용 - 여러줄일 때 교차축 방향으로 배치 (stretch 기본값, flex-start, flex-end, center, space-between, space-around)

  • 플렉서블 박스 order 적용 - 아이템 배치 순서 바꾸기 (0 기본값, 정수값)

  • 플렉서블 박스 flex 적용 - 플렉스 아이템 사이즈 조절 ([flex-grow] [flex-shrink] [flex-basis] 0 1 auto)
